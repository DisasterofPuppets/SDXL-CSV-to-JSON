SDXL CSV to JSON Converter for Fooocus

Instructions

This tool is intended to take a style.csv file used in Stable Diffusion
and convert it to .Json format for use in Fooocus (or any other tools that use the format)


1. Backup your Fooocus\sdxl_styles\sdxl_styles_marc_k3nt3l.json file

2. Copy the output file to Fooocus\sdxl_styles\sdxl_styles_marc_k3nt3l.json

3. Restart or launch Foocus


        
        

            
Don't forget to backup your old sdxl_styles_marc_k3nt3l.json file
